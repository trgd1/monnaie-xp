Méthode avec serveur local
PWA complète, installation propre Android + iPhone
›
1
Sur le PC, ouvre un terminal dans le dossier du fichier.
2
Lance : python3 -m http.server 8080 (ou double-clique LANCER_WINDOWS.bat).
3
Note l'adresse IP de ton PC (ex: 192.168.1.42).
4
Sur le téléphone (même WiFi) : ouvre http://192.168.1.42:8080/index.html dans Chrome ou Safari.
5
Installe depuis le menu du navigateur. Une fois installée, l'app fonctionne hors ligne même sans le PC.
Cette méthode active le Service Worker (cache hors-ligne), le manifest et l'icône personnalisée. C'est l'installation la plus propre.
